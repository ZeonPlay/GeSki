content_cut Splitter

Tool untuk memisah dan menggabungkan file skins.json dan geometry.json. Berguna untuk manajemen resource pack yang lebih terorganisir.

Fitur Utama:

  • Split file skins.json berdasarkan geometry yang digunakan
  • Split file geometry.json menjadi file terpisah per model
  • Merge beberapa file skins.json menjadi satu
  • Merge beberapa file geometry.json menjadi satu
  • Validasi geometry untuk memastikan kompatibilitas

Cara Penggunaan:

  1. Upload File:
    • Klik "Upload" untuk memilih file skins.json atau geometry.json
    • Untuk validasi, upload juga file geometry.json terkait
  2. Memisah File (Split):
    • Pilih item yang ingin dipisah dengan mencentang checkbox
    • Klik "Split Selected" untuk memisah item terpilih
    • Atau gunakan "Split All" untuk memisah semua item
    • File hasil split akan didownload otomatis
  3. Menggabung File (Merge):
    • Pilih beberapa file JSON yang akan digabung
    • Pastikan format file yang dipilih sesuai (skins/geometry)
    • Klik "Merge Selected" untuk menggabungkan file terpilih
    • Hasil penggabungan akan didownload sebagai file baru
  4. Validasi Geometry:
    • Upload file geometry.json yang ingin divalidasi
    • Sistem akan mengecek format dan struktur file
    • Pesan error akan muncul jika ada masalah

Tips:

  • Selalu backup file original sebelum melakukan split atau merge
  • Pastikan semua file yang akan di-merge memiliki format yang sama
  • Gunakan fitur validasi untuk mencegah error saat game berjalan
  • Nama file hasil split akan menggunakan format yang sesuai dengan isi file

face Skins Editor

Tool untuk mengedit properti skin seperti geometry, texture, animations, dan properti lainnya. Mendukung semua fitur skin Minecraft Bedrock.

Fitur Utama:

  • Edit semua properti skin (basic dan advanced)
  • Support untuk fitur enable_attachables dan hide_armor
  • Manajemen animasi dan attachment
  • Preview konfigurasi skin
  • Validasi input untuk mencegah error

Cara Penggunaan:

  1. Persiapan File:
    • Upload file skins.json yang ingin diedit
    • Sistem akan menampilkan daftar skin yang ada
  2. Edit Root Properties:
    • Serialize Name: Nama unik untuk pack (tanpa spasi)
    • Localization Name: Nama yang akan ditampilkan di game
  3. Manajemen Skin:
    • Tambah skin baru dengan klik "Tambah Skin"
    • Edit skin yang ada dengan klik tombol "Edit"
    • Hapus skin dengan tombol "Hapus"
  4. Properti Basic:
    • Localization Name: Nama skin di game
    • Geometry: Nama model yang digunakan
    • Texture: Path ke file texture (.png)
    • Type: Tipe skin (free/paid)
  5. Properti Advanced:
    • Hide Armor: Sembunyikan armor default
    • Enable Attachables: Aktifkan attachment
    • Animations: Daftar animasi yang digunakan
    • Attachments: Daftar attachment yang digunakan
    • Overlay: Path ke texture overlay
    • Color: Warna untuk tinting skin

Tips:

  • Gunakan nama yang konsisten untuk geometry dan texture
  • Properti advanced bisa ditambahkan sesuai kebutuhan
  • Preview perubahan sebelum menyimpan
  • Backup file original sebelum melakukan perubahan besar

translate Texts Editor

Tool untuk mengelola file terjemahan (.lang) yang digunakan untuk menampilkan nama pack dan skin dalam berbagai bahasa.

Fitur Utama:

  • Edit dan tambah terjemahan untuk pack dan skin
  • Generate file .lang dari skins.json
  • Validasi format terjemahan
  • Fitur pencarian dan filter

Cara Penggunaan:

  1. Memulai:
    • Upload file .lang yang sudah ada, atau
    • Generate dari skins.json untuk membuat file baru
  2. Format Terjemahan:
    • Pack: skinpack.PackName=Display Name
    • Skin: skin.PackName.SkinId=Display Name
    • Komentar: key=value ### komentar
  3. Mengedit Terjemahan:
    • Klik "Edit" pada terjemahan yang ingin diubah
    • Isi form dengan nilai baru
    • Tambahkan komentar jika diperlukan
    • Klik "Simpan" untuk menyimpan perubahan
  4. Menambah Terjemahan:
    • Klik "Tambah Terjemahan"
    • Isi key dan value sesuai format
    • Tambahkan komentar (opsional)
    • Klik "Simpan" untuk menambahkan

Tips:

  • Gunakan fitur generate untuk membuat template awal
  • Pastikan key terjemahan sesuai dengan skins.json
  • Manfaatkan komentar untuk dokumentasi
  • Gunakan fitur pencarian untuk edit cepat

3d_rotation Blockbench

Integrasi Blockbench web version untuk preview dan edit model 3D

Cara Penggunaan:

  1. Buka tab Blockbench
  2. Upload file geometry.json yang ingin di-preview
  3. Gunakan tools Blockbench untuk:
    • Preview model dari berbagai sudut
    • Edit bone dan cube
    • Mengatur tekstur
    • Export dalam format yang diinginkan
